# vertical_factory_productization

Skills that turn Factory3 modules into sellable/demo-ready products.

Current skills:
- `factory_demo_seed`
- `factory_module_publish_check`
- `factory_module_clone_plan`

Next planned:
- `factory_module_clone`
- `factory_remote_smoke_suite`

Use `factory_module_publish_check` before closing, deploying or selling a module.
Use `factory_module_clone_plan` before copying a dashboard/project to a new company.
