# vertical_apps4all_marketplace

Marketplace/catalog skills for Apps4All modules.

Uses the existing `platform.modulos`, `platform.companies` and
`platform.access_grants` tables so Apps4All stays the source of truth.

Initial skills:
- `apps4all_marketplace_schema_setup`
- `apps4all_marketplace_module_register`
- `apps4all_marketplace_module_list`
- `apps4all_marketplace_module_activate`
- `apps4all_marketplace_access_matrix`
