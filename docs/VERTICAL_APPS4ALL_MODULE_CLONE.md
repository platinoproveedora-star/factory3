# Apps4All Module Clone

Vertical para replicar apps vendibles Apps4All a otra empresa.

Contrato:
- Siempre plan primero.
- Reemplaza `company_id`, `project_code`, `module_code` y `schema`.
- Omite `node_modules`, `.next`, `.git`, `dist`, `build`, `__pycache__`.
- No escribe sin `confirm_clone=true`.
- No sobreescribe salvo `overwrite=true`.
