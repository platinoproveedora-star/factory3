# GPTAds4All - Data Contracts

All payloads travel as dictionaries in `context`. Skills do not import shared
Python schemas or dataclasses. If a field does not apply, keep the key and set
its value to `null`.

## ProductRef

```json
{
  "empresa_id": "EMP_DEMO",
  "product_key": "prod_demo_001",
  "product_name": "Curso de ingles online",
  "description": "Curso B2B para profesionales",
  "category": "educacion",
  "price_range": "500-1500 MXN",
  "url": null,
  "market": {
    "country": "MX",
    "language": "es",
    "audience": "profesionales 25-45"
  }
}
```

## ProductBrief

```json
{
  "empresa_id": "EMP_DEMO",
  "product_key": "prod_demo_001",
  "product_name": "Curso de ingles online",
  "description": "...",
  "category": "educacion",
  "price_range": "500-1500 MXN",
  "url": null,
  "market": {
    "country": "MX",
    "language": "es-MX",
    "audience": "profesionales 25-45"
  },
  "value_props": ["prop 1", "prop 2", "prop 3"],
  "tone": "profesional"
}
```

Allowed `tone`: `profesional`, `casual`, `urgente`.

## IntentSet

```json
{
  "product_key": "prod_demo_001",
  "intents": [
    {
      "intent_id": "int_001",
      "intent_text": "quiero aprender ingles para mi trabajo",
      "intent_type": "informacional",
      "funnel_stage": "awareness",
      "priority": 1
    }
  ]
}
```

Allowed `intent_type`: `informacional`, `comparacion`, `compra`.
Allowed `funnel_stage`: `awareness`, `consideration`, `decision`.
`priority` is integer 1-5, where 1 is highest priority.

## CampaignDraft

```json
{
  "empresa_id": "EMP_DEMO",
  "product_key": "prod_demo_001",
  "campaign_key": "camp_prod_demo_001_YYYYMMDDHHMMSS",
  "campaign_name": "Curso Ingles B2B - MX",
  "objective": "conversions",
  "daily_budget_amount": 500,
  "currency": "MXN",
  "status": "draft",
  "intent_ids": ["int_001"],
  "hint_ids": ["hint_001"]
}
```

Allowed `objective`: `conversions`, `traffic`, `leads`.

## CreativeSet

```json
{
  "campaign_key": "camp_demo_001",
  "creatives": [
    {
      "creative_id": "cre_001",
      "intent_id": "int_001",
      "headline": "...",
      "body": "...",
      "cta": "...",
      "variant": 1
    }
  ]
}
```

Limits:
- `headline` <= 60 chars
- `body` <= 200 chars
- `cta` <= 25 chars

## ExportArtifact

```json
{
  "campaign_key": "camp_demo_001",
  "format": "csv",
  "artifacts": [
    {
      "format": "csv",
      "file_path": null,
      "rows_exported": 12
    }
  ],
  "generated_at": "ISO 8601"
}
```

Allowed `format`: `csv`, `json`, `both`.

## Wrapper Keys

Each skill returns standard Factory output:

```json
{"ok": true, "data": {"contract_key": {}, "warnings": []}}
{"ok": false, "error": "message"}
```

Required wrapper keys:
- `gptads_product_brief_build` -> `data.product_brief`
- `gptads_intent_research` -> `data.intent_set`
- `gptads_context_hints_generate` -> `data.context_hint_set`
- `gptads_campaign_build` -> `data.campaign_draft`
- `gptads_creative_generate` -> `data.creative_set`
- `gptads_bulk_export` -> `data.export_artifact`

IDs (`intent_id`, `hint_id`, `creative_id`, `variant`) are generated by Python,
never by AI.
