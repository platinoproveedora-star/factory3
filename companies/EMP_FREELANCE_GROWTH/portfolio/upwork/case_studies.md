# Estoiko Lab Chat Agents

**Type:** AI chat agents and lead capture

## Client Problem
A marketing agency needs a way to sell AI chat agents to clients and capture leads from conversations.

## Solution
Built a Telegram-based AI agent system with company config, agent knowledge base, lead capture, Supabase storage and a Streamlit dashboard.

## Tools Used
Python, FastAPI, Telegram Bot, Anthropic, Supabase, Render, Streamlit

## Results
- Live Telegram webhook
- Reusable vertical_chat_agents skills
- Lead capture into sales.leads
- Dashboard for leads and conversations
- Hotel reception/sales demo agent

## Why It Matters
This project shows the ability to move from business context to a working system: data structure, automation logic, dashboard visibility and a path to iterate.

# EMP_LOGPLAT Fleet Operations System

**Type:** logistics operations bot and dashboard

## Client Problem
A freight/logistics operation needed a practical way to capture trips, expenses, payments and accounts receivable without depending on manual spreadsheets.

## Solution
Built a Factory3-based operations system for Platino Logistica with a dedicated Supabase schema, message handler skills, KPI skills and a Streamlit dashboard for trips, expenses, payments, accounts receivable and analysis.

## Tools Used
Python, Streamlit, Supabase, Render, Factory3 skills, Message handling, KPI automation

## Results
- Real production use by a freight company
- Dedicated logplat Supabase schema
- Dashboard sections for Overview, Viajes, Gastos, Pagos, CXC and Analisis
- Message handler skill for operational capture
- KPI skill for weekly profit, receivables balance and trip metrics
- Strong case study candidate because the system is already used in daily operations

## Why It Matters
This project shows the ability to move from business context to a working system: data structure, automation logic, dashboard visibility and a path to iterate.
